## 🔧 Customization Guide

### Change Language

**Replace Vietnamese → English**

Search files for:
```javascript
'Thêm nhóm lọc'       → 'Add Filter Group'
'Đặt lại'             → 'Reset'
'Áp dụng bộ lọc'      → 'Apply Filter'
'Lưu'                 → 'Save'
'Tải'                 → 'Load'
'Xuất'                → 'Export'
'Nhập'                → 'Import'
```

### Add New Property

1. Edit **ifc-properties.js** (line ~30):
   ```javascript
   this.propertyOptions.push('New Property');
   ```

2. Update **ifc-properties.js** extraction (line ~130):
   ```javascript
   'New Property': this.getProperty(ifcObject, 'IFC_Attribute_Name')
   ```

3. Rebuild filter engine to recognize new property

### Change Primary Color

Edit **style.css** root variables:
```css
--primary-color: #0088FF;    /* Change this */
--primary-dark: #0066CC;
--primary-light: #1A9FFF;
```

### Modify Conditions

1. Add to **ifc-properties.js** (line ~40):
   ```javascript
   this.conditionOptions.push('Custom Condition');
   ```

2. Add logic to **filter-engine.js** (line ~120):
   ```javascript
   case 'Custom Condition':
       return this.evaluateCustomLogic(objectValue, filterValue);
   ```

## 🔌 API Integration Points

Trimble Connect Viewer API methods used:

```javascript
// Required methods
viewer.setObjectColor(guid, color)      // Set object color (hex)

// Optional/custom

```

## 🐛 Known Limitations

1. **localStorage Limit**: ~5-10MB per domain
   - Each saved filter: ~1-5KB
   - Supports ~1000+ filter sets

2. **Object Property Extraction**:
   - Requires standard IFC property mapping
   - Custom properties need manual mapping

3. **Viewer API Dependency**:
   - Requires compatible Trimble Connect Viewer
   - May need API adaptation for different versions

4. **UI Limits**:
   - ~20 filter groups recommended for UI performance
   - Scrolling handles larger sets
